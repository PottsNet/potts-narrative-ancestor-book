# Changelog

## 0.8.1 - 2026-07-05

### Release readiness

- Added public GitHub metadata for version checks via `customModuleLatestVersionUrl()`.
- Updated the support URL to the public GitHub repository.
- Removed AI attribution from the public module author field.
- Updated the README for public release, manual installation and Custom Module Manager readiness.
- Added a GPL licence file.
- Added a GitHub Actions release-packaging workflow.
- Tidied public-facing wording in the module interface.

## 0.8.0

- Current module version supplied for GitHub/public-release preparation.
- Includes personalised introduction and conclusion options.
- Includes reader-friendly generation headings such as **Your Parents**, **Your Grandparents** and **Your Great-grandparents**.
- Includes HTML download support for opening the book in Microsoft Word.

## 0.7.0

- Added optional introduction and conclusion sections.
- Introduction/conclusion can be personalised, generic or turned off.
- Replaced generation headings such as **Generation 2** with reader-friendly headings.
- Updated the table of contents and downloaded HTML to use the new section names.

## 0.6.2

- Improved place wording for street addresses, schools, churches, hospitals and cemeteries.
- Shortened state-only place names where appropriate.
- Improved school, education, occupation, community service, medal and newspaper item wording.
- Hid personal additional-note blocks unless Research detail is selected.
- Avoided empty occupation facts and unknown-spouse marriage wording.

## 0.6.1

- Improved place shortening while retaining context for important locations.
- Improved migration wording.
- Suppressed empty death facts.
- Improved placement of children with no birth date.

- Filtered more source-only notes from the narrative.

## 0.6.0

- Cleaned GEDCOM note continuation markers.
- Normalised full month names.
